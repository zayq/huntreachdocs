---

## 🛠️ Étape 1 : Accéder au Microsoft Admin Center

1. Connectez-vous avec votre compte administrateur sur :  
   👉 [Microsoft Admin Center](https://admin.microsoft.com/Adminportal)
2. Allez dans la section **Utilisateurs actifs** :  
   👉 [Active Users](https://admin.microsoft.com/adminportal/home#/users)

---

## ⚙️ Étape 2 : Activer les apps de messagerie

1. Cliquez sur l’utilisateur concerné pour ouvrir le panneau latéral.
2. Allez dans l’onglet **Mail**, puis cliquez sur **Manage email apps**.
3. Activez les options suivantes :
   - ✅ **IMAP**
   - ✅ **Authenticated SMTP**
4. Cliquez sur **Save changes** pour enregistrer.

> ⚠️ **Important** : Ces options sont nécessaires pour autoriser une connexion SMTP/IMAP via HuntReach.

---

## ⏳ Étape 3 : Attendre l’activation

- Patientez environ **1 heure** pour que les paramètres soient appliqués.
- Ensuite, retournez sur HuntReach pour connecter votre compte Outlook/Microsoft 365.

---

## 🔗 Étape 4 : Connecter le compte sur HuntReach

1. Accédez aux paramètres email :  
   👉 [Paramètres Email HuntReach](https://app.huntreach.com/dashboard/settings/emails)
2. Cliquez sur **Connect Email**.
3. Choisissez l’option **Custom Email**.
4. Renseignez les champs SMTP/IMAP suivants :

| Champ         | Valeur recommandée                                |
| ------------- | ------------------------------------------------- |
| **Email**     | Votre adresse email Microsoft 365/Outlook         |
| **IMAP Host** | `outlook.office365.com`                           |
| **IMAP Port** | `993`                                             |
| **SMTP Host** | `smtp.office365.com`                              |
| **SMTP Port** | `587`                                             |
| **Password**  | Le mot de passe du compte (ou mot de passe d'app) |

> 🔐 **Sécurité** : Si la double authentification est activée, vous devrez générer un **mot de passe d’application** dans votre compte Microsoft.
