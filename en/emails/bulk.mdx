## üìù Step 2: Fill out the file

Fill in the columns as follows:

| Column           | Description                                         |
| ---------------- | --------------------------------------------------- |
| **Email**        | Address to connect (e.g., `thomas@huntreach.net`)   |
| **Display Name** | Sender name shown (e.g., `Thomas from HuntReach`)   |
| **Password**     | Account password or application-specific password   |
| **IMAP Host**    | IMAP server (e.g., `imap.gmail.com`)                |
| **IMAP Port**    | Secure IMAP port (e.g., `993`)                      |
| **SMTP Host**    | SMTP server (e.g., `smtp.gmail.com`)                |
| **SMTP Port**    | Secure SMTP port (e.g., `465`)                      |
| **Warmup**       | TRUE / FALSE depending on whether warmup is enabled |
| **Signature**    | (Optional) Custom HTML signature or leave blank     |
